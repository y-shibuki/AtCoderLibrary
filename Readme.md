# AtCoderで快適にコーディングを行うために
## 初期設定
atcoderのユーザー名とパスワードを登録  
```bash
$ oj login -u ユーザー名 -p パスワード "https://atcoder.jp/"
$ oj login --check "https://atcoder.jp/"
```

### 設定ファイルの場所
```/Users/{home}/Library/Preferences/atcoder-cli-nodejs```

### 新しい問題・テストファイルのインストール
```acc new "id"```

### ショートカットキー
・Command + Control + S  
atcoderに提出する

・Command + Control + T  
コードをテストする
